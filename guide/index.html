安装 ​

使用包管理器 ​

建议您使用包管理器 (pnpm ， yarn，npm) 安装 plus-three。

sh
pnpm install  plus-three
1
sh
yarn add  plus-three
1
sh
npm install  plus-three   --save
1

浏览器直接引入 ​

直接通过浏览器的 HTML 标签导入 plus-three，然后就可以使用全局变量 PlusProComponents 了。

根据不同的 CDN 提供商有不同的引入方式， 我们在这里以unpkg 和 jsDelivr 举例。 你也可以使用其它的 CDN 供应商。

unpkg ​

html
<head>
   <!--导入plus-three"  -->
   <script src="//unpkg.com/plus-three"></script>
</head>
1
2
3
4

jsDelivr ​

html
<head>
  <!--导入plus-three"  -->
  <script src="//cdn.jsdelivr.net/npm/plus-three"></script>
</head>
1
2
3
4

注意

默认使用最新版本，使用时建议加上版本号 如使用0.0.1版本，防止因版本导致应用出现问题。

https://cdn.jsdelivr.net/npm/plus-three@0.0.1/index.min.js
